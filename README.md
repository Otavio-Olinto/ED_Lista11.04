# ED_Lista11.04
## Fila de impressão

Este desafio consiste em criar uma aplicação, em Java, que controle uma fila de Impressão de um departamento com Múltiplos computadores.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A fila deve ser importada de uma biblioteca própria(disponível em: https://github.com/Otavio-Olinto/ED_Lista09.03);
3. A impressão deve ser apresentada como: [#PC: ID_PC – Arquivo: Nome_Arquivo];
4. Os arquivos devem seguir o padrão: ID_PC;Nome_Arquivo;
5. A aplicação deve possuir um menu de navegação.
```

```
----------------------
Menu
----------------------
1. Inserir Documento
2. Imprimir
3. Sair
----------------------
```
